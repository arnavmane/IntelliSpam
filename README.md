# IntelliSpam 

**IntelliSpam** is a Machine Learning-powered web application built with Streamlit that  classifies email text as **Spam** or **Not Spam**. 

**Try the app here:** [IntelliSpam](https://traitora.streamlit.app/)

## 🛠️ Tech Stack
- **Python:** Core programming language.
- **Streamlit:** Web framework used for the user interface.
- **scikit-learn / joblib:** For training and loading the machine learning model.
- **Pandas / NumPy:** For data handling.
- **Seaborn / Matplotlib / Wordcloud** For data visualization.

## 📂 Project Structure
- `main.py`: The Streamlit web application script.
- `EmailClassifier.ipynb`: Jupyter notebook used to explore the dataset and train the model.
- `emails.csv`: The dataset used for training the model.
